def cachesimulator.get_ram_block (   dec_ram_address)

This function takes in the search address and returns a list with the 8 byte block in ram surrounding the address as well as what the offset will be to access the specified data.

Parameters
[in]dec_ram_addressThe decimal ram address to pull the ram block from.
Returns
An 8-byte block containing the specified ram address.

◆ main()

def cachesimulator.main ( )

This is the main function that runs the entire program.


It takes a command line argument to obtain the file path to the ram input data. This input data is then used to configure the ram. Then user input is used to configure the cache. The user can then request different actions to be performed on the cache.
